# Cinestresn

## Project Overview
Cinestresn is a JavaScript-based movies streaming web application that allows users to watch their favorite films online. The application provides a user-friendly interface and supports various features to enhance the viewing experience.

## Features
- **User Authentication**: Users can sign up, log in, and manage their profiles.
- **Movie Catalog**: Browse through an extensive library of movies.
- **Search Functionality**: Easily find movies using the search bar.
- **Watchlist**: Users can add movies to their watchlist for quick access later.
- **Reviews and Ratings**: Users can leave reviews and rate movies.
- **Responsive Design**: The application is designed to work seamlessly on both desktop and mobile devices.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/bunny123373/site.git
   ```
2. Navigate to the project directory:
   ```bash
   cd site
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Usage
1. Open your web browser.
2. Navigate to `http://localhost:3000`.
3. Sign up or log in to start streaming movies.
